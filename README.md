# E-Commerce Website

A modern and responsive E-Commerce Website built using HTML, CSS, JavaScript, and Tailwind CSS. This project provides a complete shopping experience with product browsing, search, cart management, wishlist functionality, pagination, theme switching, and local storage support.

## Features

* Responsive UI Design
* Product Listing
* Product Search
* Shopping Cart
* Wishlist Functionality
* Pagination
* Product Details View
* Local Storage Integration
* Real-Time Date & Time Display
* Mobile-Friendly Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Tailwind CSS
* Font Awesome

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/E-commerce-website.git
```

2. Open the project folder.

3. Run `index.html` in your browser.

## Key Functionalities

* Add products to cart
* Remove products from cart
* Add products to wishlist
* Search products instantly
* Switch between dark and light themes
* Persistent data using Local Storage
* Responsive design for all devices
